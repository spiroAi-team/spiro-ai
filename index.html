Spiro AI

Breathing Life into AI

About the Project

Spiro AI is an innovative healthcare solution designed to analyze lung function using sensor-based spirometry and artificial intelligence. The system evaluates airflow and lung volume parameters to support early detection of respiratory disorders.

Problem Statement

Respiratory diseases such as asthma and COPD often remain undiagnosed due to limited access to affordable and accurate spirometry tools. Conventional systems are costly and require skilled operators.

Our Solution – Spiro AI

Smart Sensors

Collect real-time airflow and breathing patterns.

Mathematical Processing

Applies airflow and lung volume formulas on raw data.

AI Analysis

AI model interprets data and predicts respiratory health.

How It Works

1. User performs breathing test using sensor device
2. Airflow data is captured in real time
3. Lung parameters such as FVC and FEV are calculated
4. AI model analyzes results
5. Health insights are displayed instantly

Technologies Used

Airflow Sensor
Microcontroller
Python & Machine Learning
HTML & CSS
QR Code Integration

Future Scope

The project can be extended into a mobile application, cloud-based health records, hospital integration, and remote respiratory monitoring using IoT and AI technologies.
